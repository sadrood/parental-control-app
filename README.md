# 守护星 - 家长控制应用

一个功能完善的Android家长控制应用，包含家长端、儿童端和服务端三部分。

## 项目简介

守护星是一款专为家长设计的移动设备管理工具，帮助家长监控和管理孩子的设备使用情况，培养健康的使用习惯。

### 核心功能

#### 家长端功能
- ✅ **设备绑定管理** - 支持绑定多台儿童设备
- ✅ **应用时长限制** - 为每个应用设置每日使用时长，超时自动锁定
- ✅ **远程设备锁定** - 一键远程锁定/解锁儿童设备
- ✅ **定时任务设置** - 设置时间段自动锁定（如睡觉时间）
- ✅ **SOS紧急求助接收** - 实时接收孩子的紧急求助信号
- ✅ **网页内容过滤** - 设置禁止访问的网站
- ✅ **使用统计分析** - 查看设备使用统计数据
- ✅ **白名单应用** - 学习模式下只允许白名单应用

#### 儿童端功能
- ✅ **后台守护服务** - 前台服务保活，防止被系统清理
- ✅ **应用使用监控** - 无障碍服务监控应用使用情况
- ✅ **自动锁定机制** - 达到使用限制时自动锁定设备
- ✅ **SOS一键求助** - 紧急情况时发送位置给家长
- ✅ **锁屏保护界面** - 全屏锁定界面，防止退出
- ✅ **开机自启动** - 设备重启后自动启动守护服务

#### 服务端功能
- ✅ **WebSocket实时通信** - 家长端和儿童端实时消息推送
- ✅ **RESTful API接口** - 完整的设备管理API
- ✅ **SQLite数据存储** - 设备信息、使用统计、SOS记录
- ✅ **消息转发机制** - SOS求助实时转发给家长端

## 技术栈

### Android端
- **语言**: Java
- **最低SDK**: API 26 (Android 8.0)
- **目标SDK**: API 34 (Android 14)
- **架构**: MVVM
- **数据库**: Room (SQLite封装)
- **网络**: WebSocket (org.java-websocket)
- **UI**: Material Components + RecyclerView

### 服务端
- **运行环境**: Node.js >= 14.0
- **Web框架**: Express.js
- **WebSocket**: ws
- **数据库**: SQLite3
- **其他**: CORS, body-parser

## 快速开始

### 1. 服务端部署

```bash
cd server
npm install
npm start
```

服务将运行在 `http://localhost:3000`

### 2. 家长端编译

```bash
cd parent
./gradlew assembleDebug
```

生成的APK位于 `parent/app/build/outputs/apk/debug/app-debug.apk`

### 3. 儿童端编译

```bash
cd child
./gradlew assembleDebug
```

生成的APK位于 `child/app/build/outputs/apk/debug/app-debug.apk`

## 使用指南

### 儿童端设置步骤

1. **安装APK** - 在孩子的设备上安装儿童端APK
2. **获取绑定信息** - 打开应用，记录设备ID和绑定码
3. **启动守护服务** - 点击开关启动守护服务
4. **授予权限** - 按照提示授予以下权限：
   - 悬浮窗权限 - 用于显示锁定界面
   - 无障碍权限 - 用于监控应用使用
   - 忽略电池优化 - 防止服务被系统清理
   - 自启动权限 - 确保设备重启后服务自动启动

### 家长端设置步骤

1. **安装APK** - 在家长设备上安装家长端APK
2. **绑定设备** - 点击"绑定设备"，输入儿童端的设备ID、设备名称和绑定码
3. **设置应用限制** - 进入"应用限制"，添加需要限制的应用和每日使用时长
4. **设置定时任务** - 进入"定时任务"，设置自动锁定时间段（如睡觉时间）
5. **开始守护** - 返回首页，查看设备在线状态和使用情况

## 核心功能使用

### 应用时长限制

1. 家长端进入"应用限制"页面
2. 点击"添加限制"按钮
3. 输入应用包名（如 `com.tencent.mm`）和应用名称
4. 设置每日使用时长（分钟）
5. 可选择是否加入白名单（学习模式可用）
6. 保存后规则将自动同步到儿童端

### 定时任务

1. 家长端进入"定时任务"页面
2. 点击"添加任务"按钮
3. 设置任务名称、开始时间、结束时间
4. 选择任务类型：
   - 锁定模式：该时间段设备完全锁定
   - 学习模式：该时间段只允许白名单应用
5. 选择重复日期
6. 保存后任务将自动执行

### SOS紧急求助

儿童端有两种方式发送SOS：
1. 主界面点击"SOS紧急求助"按钮
2. 连续按电源键3次（需在设置中开启）

家长端将收到实时通知，包含孩子的位置信息。

## API接口文档

详见 `doc/API文档.md`

### 主要接口

- `GET /api` - 服务状态检查
- `POST /api/devices/bind` - 绑定设备
- `GET /api/devices` - 获取设备列表
- `POST /api/devices/:deviceId/lock` - 锁定设备
- `POST /api/devices/:deviceId/unlock` - 解锁设备
- `GET /api/sos` - 获取SOS记录列表

## 常见问题

### Q: 儿童端服务经常被系统杀死？

A: 请确保在系统设置中：
1. 关闭守护星应用的电池优化
2. 添加到自启动白名单
3. 锁定应用在后台

### Q: 如何获取应用的包名？

A: 可以通过以下方式获取：
1. 在儿童端设置-应用管理中查看
2. 使用第三方应用如"包名查看器"
3. 通过ADB命令：`adb shell pm list packages`

### Q: 锁定界面无法退出？

A: 锁定界面设计为无法通过常规方式退出，只能通过：
1. 家长端远程发送解锁命令
2. 等待定时任务结束自动解锁
3. 重启设备（守护服务会自动重新锁定）

### Q: WebSocket连接断开怎么办？

A: 系统会自动尝试重连，通常5秒内会恢复连接。如果长时间未恢复，可以重启应用或检查网络。
