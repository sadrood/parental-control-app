## 📱 编译说明

### 家长端编译

```bash
cd parent
./gradlew assembleDebug
# 输出: parent/app/build/outputs/apk/debug/app-debug.apk
```

### 儿童端编译

```bash
cd child
./gradlew assembleDebug
# 输出: child/app/build/outputs/apk/debug/app-debug.apk
```

### 服务端启动

```bash
cd server
npm install
npm start
# 运行在 http://localhost:3000
```
